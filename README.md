#  Features!

  - Scan QR Code
  - Scan Bar Code
  - Load images containing QR Code and scan them
  - Easy to use
  - Flash light
  - Qr Code Genrator
  - Bar Code Genrator
  - Admob Integration : Intertistial and Banner Ads
  - Scan History 

### How to Edit the app : 
  - Change the Package Name 
  - Change the Admob id's with your own in values.xml
